Plugins for command line to provide git status and command completion.

### Zsh

macOS ships with zsh as the default shell (since Catalina 10.15). Recommend [Oh My Zsh](https://ohmyz.sh/) for git plugin, prompt themes, and tab completion.

The built-in `git` plugin (enabled by default in Oh My Zsh) provides aliases and completion.

[ZSH Git Prompt](https://github.com/olivierverdier/zsh-git-prompt)

### Bash

Git ships with `__git_ps1` prompt support and bash completion. Source them from your `.bashrc`:

    source /usr/share/git-core/contrib/completion/git-prompt.sh

Or use [Bash Git Prompt](https://github.com/magicmonty/bash-git-prompt) for a richer experience.

### Windows

[Git for Windows](https://gitforwindows.org/) includes Git Bash, and works with [Windows Terminal](https://github.com/microsoft/terminal).

For PowerShell: [PoshGit](https://github.com/dahlbyk/posh-git)
